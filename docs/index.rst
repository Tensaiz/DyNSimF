DyNSimF - Dynamic Network Simulation Framework
==============================================

``DyNSimF`` is a Python package that serves as a framework to simulate dynamic networks.
There are a lot of functions available that make it easy to implement custom models, analyse them, and then share them with others.

The aim of this framework is to design, describe, simulate, and study dynamic network models, for example:

   - Psychological models
   - Polarization (opinion dynamics)
   - Epidemics
   - Physics network models
